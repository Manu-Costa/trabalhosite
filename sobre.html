Sobre o Projeto

Uma recriação da plataforma Netflix para um trabalho de Linguagens para Internet I.

O Projeto

Este site é um clone da Netflix desenvolvido como um trabalho. O objetivo é praticar HTML, CSS e organização de páginas web.

Tecnologias

Utilizamos HTML5 para estrutura, CSS para estilização e organização visual no estilo streaming moderno.

Design

Inspirado na interface da Netflix, com foco em layout escuro, cards e navegação simples.
